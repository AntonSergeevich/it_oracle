# README — Игровой тест личности (Big Five) для праздника

Кратко: набор файлов для лёгкой, шуточной викторины по модели Big Five. Быстрое развлечение — 10 вопросов по умолчанию, можно менять количество. В комплекте: `questions_bank.py`, `bot.py` (раннер/класс сессии), `requirements.txt`. Опционально: папка `images/` с картинками‑сценками и `results.json` для сохранённых результатов.

---

### Содержание репозитория
- **`questions_bank.py`** — банк из 15 вопросов, метки шкал, подсказки для картинок и наборы комплиментов.  
- **`bot.py`** — основной раннер и класс `QuizSession` для интеграции в бота; консольный интерфейс для тестирования.  
- **`requirements.txt`** — зависимости (aiohttp, aiofiles, beautifulsoup4, Pillow).  
- **`results.json`** — файл, куда сохраняются результаты участников (создаётся автоматически).  
- **`images/`** — (опционально) папка с картинками/сценками; имена и промпты указаны в `questions_bank.py`.  
- **`prepare_images.py`, `build_quiz.py`** — (опционально) вспомогательные скрипты для подготовки и упаковки изображений (если используешь).
